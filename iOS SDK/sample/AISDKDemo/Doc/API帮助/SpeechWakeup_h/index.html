cancelOfflineWakeup

取消本次语音唤醒识别流程。 

- (int)cancelOfflineWakeup;
Return Value

0：ok，other：fail。 错误码定义见AISDK_ERROR_*常量

initOfflineWakupWithPath:

唤醒初始化，只需要初始化一次。 

- (int)initOfflineWakupWithPath:(const char *)modelPath;
Parameters
modelPath

唤醒词模型路径

Return Value

0：ok，other：fail。 错误码定义见AISDK_ERROR_*常量

inputOfflineWakeupAudioData:withLength:

输入录音数据 

- (int)inputOfflineWakeupAudioData:(char *)data withLength:(int)dataLen;
Parameters
data

录音数据存储区域

dataLen

录音数据长度

Return Value

0：ok，other：fail。 错误码定义见AISDK_ERROR_*常量

WARNING:

建议每次调用输入的录音数据长度为4096字节。

startOfflineWakupWithData:withLength:withFlags:

开始一次唤醒识别 

- (int)startOfflineWakupWithData:(void*)userData withLength:(int)len 
        withFlags:(int)flags;
Parameters
userData

用户自定义数据， callback时将会原样带回。

len

用户自定义数据长度

flags

控制标志，参考AISDK_FLAG_WAKEUP_*常量定义，支持多flag或运算。如果不设置，传0即可。

Return Value

0：ok，other：fail。 错误码定义见AISDK_ERROR_*常量

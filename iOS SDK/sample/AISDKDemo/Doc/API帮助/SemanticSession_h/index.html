Introduction

语义识别能力提供者，该类封装了语义能力的所有接口


Protocols

SemanticSessionDelegate

语义识别回调接口，在SemanticSession对象中注入


Methods

+cancelSemanticByType:

根据请求类型，取消特定语义请求

+cancelSemanticByUserData:

根据外部传入数据，取消特定语义请求

+cancelText2Semantic

取消本次语义请求

+complexSemantic2Semantic:userData:

特殊功能性接口，复合语义

+reqFM:showType:extraMsg:

请求单个FM的详细信息，播放FM时，FM的地址可能会有变化，播放前应该用FM的ID请求一次新的url

+reqMusic:extraMsg:

请求单首音乐的详细信息，播放音乐时，音乐的地址可能会有变化，播放前应该用音乐的ID请求一次新的url

+text2semantic:userData:flags:

在线二次识别接口，结果通过回调异步回调返回iption

flags

控制标志，参考K_AISDK_FLAG_SEMANTIC_*常量定义。默认设置为0即可

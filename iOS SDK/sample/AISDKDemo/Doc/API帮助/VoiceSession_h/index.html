+cancelVoice2Text

取消本次语音识别

+inputVoice2TextAudioData:

输入录音数据

+stopVoice2Text

停止本次语音识别，获取语音识别结果。

WARNING:

停止后，不会中断语音识别，SDK会将现在已经接收到的音频发送到后台请求语音识别结果。

cancelVoice2Text

取消本次语音识别 

-(int)cancelVoice2Text;
Return Value

0：ok，other：fail。 错误码定义见AISDK_ERROR_*常量

inputVoice2TextAudioData:

输入录音数据 

-(int)inputVoice2TextAudioData:(NSData *)audioBuffer;
Parameters
audioBuffer

上传音频数据

Return Value

0：ok，other：fail。 错误码定义见AISDK_ERROR_*常量 @warning: 上层应用应当把实时录音数据通过该接口输入到SDK（建议每次输入4096字节长度的音频）。 在调用此接口过程中，会有VAD检测到开始、VAD检测到结束、实时音量、语音识别到结果等 状态的回调。遇到AISDK_CMD_ONLINE_RECO_CANCELED/AISDK_CMD_ONLINE_RECO_TIME OUT/AISDK_CMD_ONLINE_RECO_ERROR/AISDK_CMD_ONLINE_RECO_RESULT时，停止调用 该函数。

stopVoice2Text

停止本次语音识别，获取语音识别结果。

WARNING:

停止后，不会中断语音识别，SDK会将现在已经接收到的音频发送到后台请求语音识别结果。 

-(int)stopVoice2Text;
Return Value

0：ok，other：fail。 错误码定义见AISDK_ERROR_*常量

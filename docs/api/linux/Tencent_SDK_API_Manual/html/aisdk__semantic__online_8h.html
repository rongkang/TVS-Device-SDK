AISDK_API_EXPORTS int aisdkOnlineText2Semantic ( const char *  text,
int  textLen,
void *  userData,
int  len,
int  flags 
)

在线语义识别接口， 结果通过回调异步回调返回

参数
text以utf8格式编码的文本串，以'\0'结尾
textLen文本串长度
userData自定义信息，callback时带回。
len自定义信息长度
flags控制标志，参考AISDK_FLAG_SEMANTIC_*常量定义，支持多flag或运算。默认设置为0即可。
返回
0：ok，other：fail。 错误码定义见AISDK_ERROR_*常量
注解
获取"上一首/下一首"等.
